8-Step Automated Pipeline

全自动 8 步处理流水线

From raw EMDB entries to training-ready annotated data — fully automated, quality-controlled, and reproducible.

从原始 EMDB 条目到可直接用于训练的标注数据，全程自动化、质量可控、完全可复现。

STEP 01
⬇️
Retrieval
数据下载
Download density maps from EMDB FTP and atomic models from RCSB PDB. Auto-fetch resolution metadata.
从 EMDB FTP 下载密度图，从 RCSB PDB 下载原子模型，自动获取分辨率元数据。
STEP 02
🔄
Resampling
重采样
Resample density maps to uniform 1.0 Å/voxel using scipy interpolation. Handle non-standard MRC axis orders.
用 scipy 插值将密度图重采样到统一 1.0 Å/voxel，正确处理非标准 MRC 轴排列。
STEP 03
📊
Normalization
归一化
98th-percentile normalization on non-zero voxels. Clips to [0, 1] for consistent training input.
对非零体素取第98百分位归一化，输出值域 [0,1]，保证训练输入的一致性。
STEP 04
🔬
Alignment QC
对齐质控
PHENIX map_model_cc for CC_mask/CC_volume/CC_peaks. Per-atom Q-score using Pintilie 2020 method.
PHENIX 计算 CC_mask/CC_volume/CC_peaks，Pintilie 2020 方法逐原子计算 Q-score 置信度。
STEP 05
🧩
Domain Segmentation
结构域分割
Merizo deep learning model segments each chain into structural domains. Propagated to bio-assembly copies.
Merizo 深度学习模型将每条链分割为结构域，并传播到生物组装体的所有对称副本。
STEP 06
🏷️
Voxel Labeling
体素标注
8-channel CA-only annotation paradigm. Three KDTree groups for semantic labels at atomic resolution.
8 通道 CA-only 标注范式，三组 KDTree 在原子分辨率下生成语义标签。
STEP 07
✨
Simulated Density
模拟密度图
ChimeraX molmap generates ideal noiseless density map from atomic model — the perfect denoising target.
ChimeraX molmap 从原子模型生成理想无噪声密度图，作为去噪任务的完美目标。
STEP 08
🏆
Tier Classification
质量分级
Gold / Silver / Bronze / Reject tiers based on resolution, CC_mask, and Q-score with resolution-binned thresholds.
按分辨率区间设阈值，依据 CC_mask 和 Q-score 分为 Gold/Silver/Bronze/Reject 四档。

每个条目综合 CC_mask（PHENIX）、Q-score 密度拟合置信度和分辨率进行分级， 下游任务可按质量筛选数据。

Scientific Impact
科学价值

Enabling Next-Generation Research

赋能下一代研究

This dataset serves as the foundation for a new class of deep learning models targeting cryo-EM structural biology.

该数据集是新一代深度学习模型的基础，专为冷冻电镜结构生物学设计。

🎯
Map-Model Alignment
密度图—模型对齐
8-channel voxel annotations enable end-to-end supervised learning for automatic map-model fitting — the primary dataset objective.
8 通道体素标注使端到端监督学习自动密度图—模型拟合成为可能，这是本数据集的核心目标。
🔇
Denoising & Enhancement
去噪与增强
Paired experimental and ChimeraX simulated density maps provide ground truth for map denoising and resolution enhancement models.
配对的实验密度图与 ChimeraX 模拟密度图，为密度图去噪和分辨率增强模型提供真值。
🧬
Protein Interface Prediction
蛋白质界面预测
Interface labels derived from CA-CA distance thresholding enable training of interface-aware models directly from density data.
基于 CA-CA 距离阈值的界面标注，支持直接从密度图数据训练界面感知模型。
🏗️
Domain Architecture Learning
结构域架构学习
Merizo-derived domain labels propagated across bio-assemblies enable density-based domain segmentation without atomic models.
Merizo 生成并传播到生物组装体的结构域标注，支持无需原子模型的密度图结构域分割。
⚖️
Quality-Aware Training
质量感知训练
Gold/Silver/Bronze tiers with PHENIX CC metrics allow curriculum learning — start with high-quality data, gradually add harder examples.
PHENIX CC 指标支撑的 Gold/Silver/Bronze 分级，支持课程学习：从高质量数据开始，逐步加入难例。
📐
Secondary Structure in Density
密度中的二级结构
Helix/Strand/Coil labels at CA positions enable density-based secondary structure prediction — bridging cryo-EM and NLP-style SSP.
CA 位置的螺旋/折叠/无规则卷曲标注，使基于密度图的二级结构预测成为可能。
